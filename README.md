# Geography Quiz

## Overview

Geography Quiz is a C# desktop application that tests users' knowledge of countries, capitals, flags, and general geography through an interactive quiz interface.

## Features

- Multiple-choice questions
- Score tracking
- Instant feedback
- User-friendly interface

## Technologies Used

- C#
- Windows Forms
- Visual Studio

## Installation

1. Clone the repository.
2. Open the solution in Visual Studio.
3. Build and run the application.

## Future Improvements

- Difficulty levels
- Leaderboard
- Question database expansion
