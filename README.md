# 🌍 Geography Quiz App

A C# console-based Geography Quiz application developed using Object-Oriented Programming (OOP) principles. The application provides an interactive quiz experience with multiple question types, allowing users to create, manage, and play geography quizzes while reviewing corrections for incorrect answers.

---

## 📌 Features

- Multiple Choice Questions
- Open-Ended Questions
- True/False Questions
- Create new quiz questions
- Edit existing questions
- Delete questions
- Interactive quiz gameplay
- Automatic score calculation
- View corrections for incorrect answers
- User-friendly console interface with colored output

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/Geography-Quiz-App.git
```

2. Open the project in Visual Studio.

3. Build and run the application.

Or using the .NET CLI:

```bash
dotnet run
```
